Benefits:

  📊 Real-time streaming
  🚀 Faster than SQLite for reads

Enable Redis:
  $ sudo apt install redis-server    # Ubuntu/Debian
  $ brew install redis               # macOS
  $ redis-server                     # Start Redis
  
  Edit .3ox/config.toml:
    [redis]
    enabled = true
    host = "localhost"
    port = 6379

NOCTUA will auto-detect and use Redis if available.
Falls back to SQLite-only mode if not.

┌─────────────────────────────────────────────────────────────────────────┐
│ 🧪 API & EXTENSIBILITY                                                  │
└─────────────────────────────────────────────────────────────────────────┘

Python API Example:
  from owl import NoctuaOwl
  
  owl = NoctuaOwl()
  owl.set_memory("project", "NOCTUA")
  value = owl.get_memory("project")
  owl.log_history("user", "Hello!")
  
  block_id = owl.create_folder_block(
      folder_path="/workspace",
      block_type="data",
      config={"name": "Workspace"},
      position=(200, 150)
  )
  
  blocks = owl.get_folder_blocks()
  owl.backup()
  owl.shutdown()

REST API (Dashboard):
  GET  /api/blocks           - List all blocks
  POST /api/blocks           - Create new block
  POST /api/blocks/save      - Update positions
  GET  /api/stats            - System statistics
