🎨 MESSAGE RITUAL FORMAT

Sealed Message (persists to database):

    Banner Line — Auto-extracted first line
    Your content goes here.
    Multiple lines are supported.
    :: ∎

Unsealed Message (ephemeral):

    Regular message without seal.
    Cached in Redis but not persisted.

═══════════════════════════════════════════════════════════════════

🔑 KEY CONCEPTS

Banner:     First line of message (auto-extracted)
Seal:       Marker :: ∎ (indicates persistence)
Events:     All interactions logged with timestamps
Facts:      Key-value storage with confidence scores
Chunks:     Text blocks ready for embeddings
Drift-lock: Structure preserved, no decay
