SECURITY: ⟦⎊⟧ as AUTH TOKEN

The Unicode marker ⟦⎊⟧ serves two purposes:

1. LOAD TESTING — Breaks weak parsers (diagnostic feature)
2. AUTH TOKEN — Part of SHA256 validation hash

Generate auth key:
  export CODEX_SECRET="my-secret"
  echo -n "///▙⟦⎊⟧::∎my-secret" | sha256sum

Add to spec:
  [ritual]
  auth_key = "computed-hash-here"

Execute:
  CODEX_SECRET="my-secret" stratos myop.toml

Fallback: Use [⎊] for ASCII-safe environments

═══════════════════════════════════════════════════════════════════

OPERATION SPEC FORMAT:

[ritual]    # Auth markers (ask, boot, seal, auth_key)
[meta]      # Metadata (name, version, operator)
[kernel]    # Execution laws (purpose, rules, identity)
[[plan]]    # Steps to execute (type, id, ...)

Step types:
  • shell     — Execute commands
  • llm       — Call language model
  • ruby      — Run Ruby script
  • r         — Run R analysis
  • validate  — Check conditions
  • store     — Save to context
  • recall    — Retrieve from context
